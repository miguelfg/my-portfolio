# <a href="#" class="u-url">Documentation</a>

Here is all the documentation for Nikola. Note that it applies to the
latest release, so it may not be accurate if you’re using an older
version, or are using an unstable version. If you want to read the
documentation for the current GitHub master, check out
<a href="http://nikola.readthedocs.org/en/latest/"
class="reference external">Nikola on ReadTheDocs</a>.

<a href="handbook.html" class="reference external">The Handbook</a>  
Aimed at those wanting to *use* Nikola to build a site.

<a href="getting-started.html" class="reference external">Getting
Started</a>  
How to set up Nikola.

<a href="conf.html" class="reference external">The conf.py file</a>  
This is the default configuration file for the latest version of Nikola.
It contains some useful documentation not mentioned in the manual, and
can be used for people upgrading to compare with their existing config
files.

<a href="blog/upgrading-to-nikola-v8.html"
class="reference external">Upgrading to Nikola v8</a>  
A blog post detailing all the changes from Nikola v7.

<a href="changes.html" class="reference external">Changelog</a>, <a href="authors.html" class="reference external">Contributors</a>, <a href="license.html" class="reference external">The MIT License</a>  
What’s new, who made this, and the rules for distribution.

<a href="creating-a-custom-page.html"
class="reference external">Creating Custom Pages</a>  
How to do custom pages for special purposes, with special templates and
styling.

<a href="creating-a-theme.html" class="reference external">Theming
Tutorial</a>, <a href="theming.html" class="reference external">Theming reference</a>, <a href="template-variables.html" class="reference external">Template

Useful if you want to create or customize a theme.

<a href="extending.html" class="reference external">Extending Nikola</a> and <a href="internals.html" class="reference external">Nikola Internals</a>  
Aimed at those wanting to add new functionality.

<a href="features/index.html" class="reference external">Features</a>  
A list of some Nikola features.

<a href="creating-a-site-not-a-blog-with-nikola.html"

Not everything is a blog!

<a href="path-handlers.html" class="reference external">Path handlers

<a href="social_buttons.html" class="reference external">Using Social

Replacing AddThis with other solutions.

<a href="blog/automating-nikola-rebuilds-with-github-actions.html"
