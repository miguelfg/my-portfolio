## Host anywhere.

[Static websites](/handbook.html#why-static) are safer, use fewer
resources, and avoid vendor and platform lock-in. You can host a Nikola
website on any web server, big or small. It’s just a bunch of HTML files
and assets.

<span class="fa-stack fa-3x"></span>

## Fast rebuilds.

[Nikola is fast.](/features/fast.html) We use
[doit](http://pydoit.org/), which provides incremental rebuilds — in
other words, we rebuild only the pages that *need* rebuilding, saving
CPU time, wall clock time and upload bandwidth.

## Multiple input formats.

[Nikola will take input in many
formats](/handbook.html#supported-input-formats). Out of the box, we
support reStructuredText, Markdown, IPython (Jupyter) Notebooks and
HTML, and have plugins for many other formats.

## Batteries included.

[Nikola comes with everything you need](/handbook.html#components) to
build a modern website: blogs (with comments, tags, categories,
archives, RSS/Atom feeds), multilingual support, easy image galleries,
and code listings.

## Easily extensible.

[Nikola is extensible](/extending.html). You can write a plugin to add
any feature you want in a few lines of Python, or write your own theme
in Mako or Jinja2. Or find something in the
[Plugin](https://plugins.getnikola.com) and
[Theme](https://themes.getnikola.com/) Indexes.

## User-friendly CLI.

[Nikola has a friendly user interface](/features/index.html#cli) that
gets you up and running quickly and simplifies your work. You do not
need to memorize headers just to create a post — we’ll write them for
you.
