# My Portfolio

Personal portfolio website built with **Nikola** (static site generator).

## Tech Stack
- **Generator**: [Nikola](https://getnikola.com/)
- **Language**: Python
- **Package Management**: [uv](https://github.com/astral-sh/uv)
- **Deployment**: GitHub Pages (Branch: `main` → `gh-pages`)

## Documentation
- **Project**: See `CLAUDE.md` and `AGENTS.md`.
- **Nikola**: Local offline documentation available in `docs/nikola-docs/`.

## Content
- **Blog**: Technical articles, guides, and project updates.
- **Portfolio**: Showcase of data analysis and engineering projects.
- **AI News**: Automated daily summaries of AI trends.

## Development
All Nikola commands run from the `portfolio/` directory using `uv run nikola ...`.
